# Document.querySelector()  querySelector() 함수
  - 괄호 속에 제공한 선택자와 일치하는 문서 내 첫 번째 Element를 반환
  - 일치하는 요소가 없다면 null 반환
  
  ** 사용 예시
  const selected = document.querySelector("h1");
  
  - 문서 내의 첫 번째 h1 태그를 찾아 selected에 반환해줍니다.
  
  ** 사용 방법 4가지
  1. 태그 select
  2. 클래스 select
  3. id select
  4. 복합 select
