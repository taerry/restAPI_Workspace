*** JSONPlaceholder
	- 웹사이트 "https://jsonplaceholder.typicode.com/"는 JSONPlaceholder에서 제공하는무료온라인REST API 서비스
	- 개발자가 프런트엔드 또는 백엔드 애플리케이션을 구축할 때 테스트 및 프로토타입 제작을 위해 주로 사용
	
	* JSONPlaceholder는 개발자가 애플리케이션을 테스트하는 데 사용할 수 있는 다양한 모의 데이터를 제공
	  - 데이터에는 다음과 같이 REST API에 있는 일반적인 리소스를 포함
	  ---------------------------------------------------------------
	  -게시물(Posts): 블로그 게시물 또는 기사입니다.
	  -댓글(Comments): 게시물과 관련된 댓글입니다.
	  -앨범(Albums): 사진 모음입니다.
	  -사진(Photos): 앨범과 관련된 이미지입니다.
	  -할 일(Todos): 작업 목록.
	  -사용자(Users): 사용자에 대한 정보입니다.
	  ---------------------------------------------------------------
	  예시 엔드포인트는 다음과 같습니다.
	  * GET /posts - 게시물 목록을 검색합니다.
	  * GET /posts/1 - ID로 특정 게시물을 검색합니다.
	  * POST /posts - 새 게시물을 만듭니다.
	  * PUT /posts/1 - 특정 게시물을 업데이트합니다.
	  * DELETE /posts/1 - 특정 게시물을 삭제합니다.
	   ---------------------------------------------------------------
