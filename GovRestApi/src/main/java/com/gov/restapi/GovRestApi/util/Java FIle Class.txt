* 자바에서는 파일과 디렉토리에 대한 정보를 가지고있는 java.io.File, java.nio.file.Files 클래스를 제공하고 있습니다. 
  Files는 File을 개선한 클래스입니다.
  
  File 객체 생성은 다음과 같이 합니다.
  -----------------------------------------
  File file = new File("파일 경로");
  
  파일 경로를 지정하는 방법은 절대 경로, 상대 경로가 있습니다.
  
  위와 같이 객체를 생성했다고 해서 없던 파일이 파로 생기거나, 바로 작업을 할 수 있는 것은 아닙니다. 
  exists() 메소드를 통해서 지정한 파일이 존재하고 있는지 아닌지를 판별하고 그 결과에 따라서 추가적인 작업을 수행하게 됩니다.
